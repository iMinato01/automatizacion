        **  Sistema Administrativo  **

El objetivo es automatizar y centralizar los procesos actuales internos para optimizar
los tiempos de trabajo.

        //  Evaluaciones de Proveedor //

Inicialmente el módulo de Evaluaciones de Proveedor permitirá extraer información de un
formulario, enviandola al servidor, procesando esta misma y relacionando un proveedor con
la evaluación, manteniendo así un control mas optimo sobre esto y  facilitando al evitar
crear multiples archivos, permitiendo exportarlo directamente a PDF.
